## Заметки

- Сохраняйте порядок задач согласованным с планом
- Используйте phase-scoped task IDs в формате `T<phase>.<index>`
- Держите задачи конкретными, измеримыми и исполнимыми по порядку
- По возможности ссылайтесь в каждой задаче на 1-2 стабильных ID (`AC-*`, `RQ-*`, `DEC-*`)
- Отмечайте задачи выполненными по мере реализации
- Не оставляйте критерии приемки без покрытия задачами
