# Demo kit

This folder contains reproducible demo assets for `draftspec`.

## Record the quick GIF

Prerequisites:

- `go`
- `vhs` (https://github.com/charmbracelet/vhs)

From the repository root:

```bash
bash demo/scripts/record-quick.sh
```

Outputs:

- `demo/draftspec-demo.gif`

Temporary workspace:

- `demo/_work/` (gitignored)
